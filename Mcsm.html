Hoe krijg ik een "ngrok" code?

Door een account te maken op dashboard.ngrok.com/signup. Als je ingelogd bent ga je naar dashboard.ngrok.com/auth. Daar vind je je code aan de bovenkant van de website.

Waarom heb ik een "ngrok" code nodig?

Ngrok zorgt ervoor dat je vrienden kunnen joinen. Als het niet aanstaat kunnen alleen mensen op jouw netwerk joinen. Maar maak je geen zorgen, als je je code hebt ingesteld gaat dat allemaal vanzelf

Hoe installeer ik dit?

ik heb hier een filmpje over hoe je alles gebruikt. LET OP! Je hoeft maar één keer op update code te klikken. Daarna hoef je alleen maar op start ngrok te klikken.
